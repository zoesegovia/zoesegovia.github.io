Tipos de selectores que entran en examenes

Class (class=””)

Selecciona todos los elementos a los que se les ponga ese atributo con un determinado texto en las comillas

Por ejemplo: si pones dentro de varias etiqueta el atributo: class:"colorear", se aplicará el estilo que elijas a todas ellas

Identificador (id=””)

Selecciona el elemento alque se le ponga ese atributo con un determinado texto en las comillas

Por ejemplo: si pones dentro de varias etiquetas el atributo: id="bonito", se aplicará el estilo que elijas a esa etiqueta

Todos los elementos (*)

Selecciona todos los elementos

Por ejemplo: si pones: *, se aplicará el estilo que elijas a todos los elementos

Varios elementos ( p, em, h1, h2)

Selecciona todos los elementos de los tipos que elijas, pueden ser: p, h1, em, h2, h3...

Por ejemplo: si pones: h1, h2, se aplicara a las etiquetas h1 y h2

Elementos contenidos en otros elementos (ol li)

Selecciona todos los elementos de un tipo contenido en otros.

Por ejemplo: si pones: h1 li, se aplicara a los elemtos li que esten dentros de los elementos h1

Sobrevuleo (p:hover)

Al pasar el ratón sobre los elementos seleccionados estos cambiarán y pasarana a adquirir las características que se le pongan
